NextGIS BIO
============

![ui](http://docs.nextgis.ru/_images/bio_common.png)

NextGIS Bio is a server based GIS application to biodiversity monitoring.

Used program components:

1. Server side
  * Python
  * Pyramid
  * PostgreSQL with PostGIS
2. Client side:
  * Dojo
  * Openlayers
